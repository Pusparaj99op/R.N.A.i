Project Overview

RescueNet AI is a comprehensive IoT-based emergency response system that monitors health vitals in real-time and automatically alerts emergency services when anomalies are detected. The system uses ESP32 as the main controller, with an optional Arduino Nano configuration, connected to various sensors and modules.

Key Components
  • ESP32: Main controller with WiFi & Bluetooth
  • Arduino Nano: Alternative/secondary controller
  • MAX30102: Heart rate & SpO2 sensor
  • MPU6050: Accelerometer/Gyroscope for fall detection
  • DS18B20: Body temperature sensor
  • SSD1306: OLED display for showing vitals
  • SIM800L: GSM module for SMS alerts

Important Power Warning

The SIM800L module requires a dedicated 3.7V-4.2V power supply. Do NOT connect to 5V as it will permanently damage the module!

ESP32 Configuration

ESP32 with sensors and modules

ESP32 Pin Usage

Power Pins:

  • 3.3V: Power for sensors
  • GND: Common ground

I2C Pins:

  • GPIO 21: SDA (Data)
  • GPIO 22: SCL (Clock)

Sensor/Component Pins:

  • GPIO 0: Emergency Button
  • GPIO 2: Buzzer +
  • GPIO 4: DS18B20 Data
  • GPIO 5: Status LED
  • GPIO 14: SIM800L RST
  • GPIO 15: SIM800L PWR
  • GPIO 16: SIM800L RXD
  • GPIO 17: SIM800L TXD
  • GPIO 18: Emergency LED

Arduino Nano Configuration

Arduino Nano with sensors and modules

Arduino Nano Pin Usage

Power Pins:

  • 5V: Power for sensors
  • GND: Common ground

I2C Pins:

  • A4: SDA (Data)
  • A5: SCL (Clock)

Digital Pins:

  • D2: Emergency Button
  • D3: DS18B20 Data
  • D4: Status LED
  • D5: Buzzer
  • D6: Emergency LED
  • D7-D8: Reserved

Serial Communication:

  • TX/RX: Communication with ESP8266

Additional Components

LEDs & Button
  • Status LED (Green):
    ESP32: GPIO 5 → 220Ω → LED → GND
    Nano: D4 → 220Ω → LED → GND
  • Emergency LED (Red):
    ESP32: GPIO 18 → 220Ω → LED → GND
    Nano: D6 → 220Ω → LED → GND
  • Emergency Button:
    ESP32: GPIO 0 ↔ Button ↔ GND
    Nano: D2 ↔ Button ↔ GND
Buzzer
  • ESP32 Configuration:
    Positive (+): GPIO 2
    Negative (-): GND
  • Arduino Nano Configuration:
    Positive (+): D5
    Negative (-): GND
Power Supply
  • ESP32:
    5V via USB or 7-12V via VIN pin
  • Arduino Nano:
    5V via USB or 7-12V via VIN pin
  • SIM800L:
    3.7V Li-Ion battery (dedicated)
    ⚠️ Do NOT use 5V!
  • Sensors:
    3.3V from ESP32 or Arduino's 3.3V regulator
